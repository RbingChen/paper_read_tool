1. 内容理解

本文档描述了 DeepSeek-GRM 模型的响应评分框架，包含两套评价标准：

  • 通用评价标准：包含准确性（Accuracy）、详细程度（Level of Detail）、相关性（Relevance）三个维度，每项按 1-10 分四级划分。
  • 指令遵循专项标准：新增指令遵循度（Instruction Adherence）、清晰度（Clarity）、准确性（Accuracy）维度，同样采用四级评分。

核心机制：根据对话上下文和查询要求，先分析潜在特定标准及权重，再基于标准对多个响应进行严格评分（0/1 二元判定）。评分需包含三个输出行：具体标准、对比分析、最终得分。

2. 内容翻译

General Evaluation Criteria

1. Accuracy:
- Completely Accurate (9-10 points): The response provides completely accurate information, fully addressing the issue.
- Useful but Incomplete (6-8 points): The response provides some useful information, but lacks details or accuracy.
- Limited Usefulness (3-5 points): The response offers little useful information, with most content being irrelevant or incorrect.
- Useless or Incorrect (1-2 points): The response is completely irrelevant or incorrect.

Example: If there are factual errors in the response but the overall direction is correct, it falls under “Useful but Incomplete.”

2. Level of Detail:
- Very Detailed (9-10 points): The response includes ample details covering all aspects of the issue.
- Detailed but Slightly Lacking (6-8 points): The response is fairly detailed but misses some important details.
- Basically Detailed (3-5 points): The response provides some details but is not thorough enough overall.
- Not Detailed (1-2 points): The response is very brief and lacks necessary details.

Example: If the response provides only a simple conclusion without an explanation, it falls under “Not Detailed.”

3. Relevance:
- Highly Relevant (9-10 points): The response is highly relevant to the question, with information closely aligned with the topic.
- Generally Relevant (6-8 points): The response is generally relevant but includes some unnecessary information.
- Partially Relevant (3-5 points): The response has a lot of content that deviates from the topic.
- Not Relevant (1-2 points): The response is completely irrelevant.

Example: If the response strays from the topic but still provides some relevant information, it falls under “Partially Relevant.”

通用评价标准

1. 准确性（Accuracy）：
- 完全准确（9-10分）：响应提供完全准确的信息，彻底解决问题。
- 有用但不完整（6-8分）：响应提供部分有用信息，但缺乏细节或准确性。
- 有限价值（3-5分）：响应提供的有用信息很少，大部分内容无关或不正确。
- 无用或错误（1-2分）：响应完全无关或不正确。

示例：若响应存在事实错误但整体方向正确，则属于“有用但不完整”。

2. 详细程度（Level of Detail）：
- 非常详细（9-10分）：响应包含涵盖问题所有方面的充分细节。
- 详细但略有不足（6-8分）：响应相当详细但遗漏重要细节。
- 基本详细（3-5分）：响应提供部分细节但整体不够全面。
- 不详细（1-2分）：响应非常简略且缺乏必要细节。

示例：若响应仅提供简单结论而无解释，则属于“不详细”。

3. 相关性（Relevance）：
- 高度相关（9-10分）：响应与问题高度相关，信息紧扣主题。
- 总体相关（6-8分）：响应基本相关但包含不必要信息。
- 部分相关（3-5分）：响应有大量偏离主题的内容。
- 不相关（1-2分）：响应完全无关。

示例：若响应偏离主题但仍提供部分相关信息，则属于“部分相关”。

Evaluation Criteria (Instruction-Focused)

1. Instruction Adherence:
- Fully Adhered: The response fully complies with all instructions and requirements of the question.
- Partially Adhered: The response meets most of the instructions but has some omissions or misunderstandings.
- Basically Adhered: The response meets some instructions, but the main requirements are not fulfilled.
- Not Adhered: The response does not meet any instructions.

Example: If the question requires three examples and the response provides only one, it falls under “Partially Adhered.”

2. Clarity:
- Very Clear: The response is fluent, well-structured, and logically clear.
- Clear but Minor Issues: The response is mostly clear but has some minor language or structural issues.
- Basically Clear: The response has noticeable language or logic issues but is still understandable.
- Not Clear: The response is disjointed, illogical, and hard to understand.

Example: If the response has complex sentence structures and lacks punctuation, it falls under “Basically Clear” or “Not Clear.”

3. Accuracy:
- Completely Accurate: All information and data are completely accurate.
- Mostly Accurate: Most information is accurate, with minor errors.
- Some Errors: There are some noticeable errors affecting comprehension.
- Mostly Incorrect: There are numerous errors seriously affecting the credibility of the information.

Example: If a specific data point is incorrectly cited but doesn’t affect the overall conclusion, it falls under “Mostly Accurate.”

评价标准（指令遵循专项）

1. 指令遵循度（Instruction Adherence）：
- 完全遵循：响应完全符合问题的所有指示和要求。
- 部分遵循：响应满足大部分指示但存在遗漏或误解。
- 基本遵循：响应满足部分指示，但主要要求未达成。
- 未遵循：响应未满足任何指示。

示例：若问题要求三个例子而响应仅提供一个，则属于“部分遵循”。

2. 清晰度（Clarity）：
- 非常清晰：响应流畅、结构良好、逻辑清晰。
- 清晰但有小问题：响应基本清晰但存在轻微语言或结构问题。
- 基本清晰：响应存在明显语言或逻辑问题但仍可理解。
- 不清晰：响应支离破碎、缺乏逻辑且难以理解。

示例：若响应句式复杂且缺少标点，则属于“基本清晰”或“不清晰”。

3. 准确性（Accuracy）：
- 完全准确：所有信息和数据完全正确。
- 基本准确：大部分信息准确，存在轻微错误。
- 部分错误：存在影响理解的明显错误。
- 基本错误：存在严重损害信息可信度的多处错误。

示例：若具体数据引用错误但不影响整体结论，则属于“基本准确”。

3. 摘要总结

本文定义了 DeepSeek-GRM 模型的双层响应评分体系：

  1. 通用标准：从准确性（Accuracy）、详细程度（Level of Detail）、相关性（Relevance）三个维度，按 1-10 分四级制评估响应质量，每项均配有分级描述和示例。
  2. 指令专项标准：强化对指令遵循度（Instruction Adherence）、清晰度（Clarity）、准确性（Accuracy）的考核，同样采用四级评分机制。
  3. 评分流程：基于对话上下文，先确定特定标准及权重，再严格对比多个响应，最终输出三项内容（具体标准说明、响应对比分析、逗号分隔的 0/1 评分结果）。

核心特点：采用严格二元判定（0/1），要求评分过程包含逐步分析，示例系统完善。

4. 术语识别

Instruction Adherence（指令遵循度）
衡量响应符合用户指令的程度。四级标准：完全遵循（满足所有要求）、部分遵循（满足主要要求但有遗漏）、基本遵循（仅满足次要要求）、未遵循（完全偏离指令）。
Accuracy（准确性）
评估响应信息的正确性。在通用标准中关注事实准确性，在指令标准中额外关注数据精确性。分级包括：完全准确（无错误）、基本准确（非关键错误）、部分错误（影响理解）、基本错误（严重失真）。
Level of Detail（详细程度）
考察响应内容的完整性与深度。四级划分：非常详细（覆盖所有方面）、详细但略有不足（遗漏次要细节）、基本详细（仅核心内容）、不详细（缺失必要细节）。
Relevance（相关性）
判定响应与主题的关联强度。分级标准：高度相关（完全聚焦主题）、总体相关（含少量冗余）、部分相关（显著偏离主题）、不相关（完全无关）。
Clarity（清晰度）
评价语言表达与逻辑结构。四级描述：非常清晰（流畅且逻辑严密）、清晰但有小问题（轻微表达瑕疵）、基本清晰（可理解但有明显缺陷）、不清晰（支离破碎难以理解）。
DeepSeek-GRM（评分模型）
专注于响应评分的AI系统，通过双层标准（通用+指令专项）对多个响应进行严格对比分析，最终输出二元评分（0/1）。
