Query Write a JavaScript function ‘const tri = (n)‘ to solve the following problem:

Everyone knows Fibonacci sequence, it was studied deeply by mathematicians in the last couple centuries. However, what people don’t know is Tribonacci sequence.

Tribonacci sequence is defined by the recurrence:

tri(1) = 3

tri(n) = 1 + n / 2, if n is even.

tri(n) = tri(n - 1) + tri(n - 2) + tri(n + 1), if n is odd.

For example:

tri(2) = 1 + (2 / 2) = 2

tri(4) = 3

tri(3) = tri(2) + tri(1) + tri(4) = 2 + 3 + 3 = 8

You are given a non-negative integer number n, you have to a return a list of the first n + 1 numbers of the Tribonacci sequence.

Examples: tri(3) = [1, 3, 2, 8]

查询 编写JavaScript函数‘const tri = (n)‘解决以下问题：

众所周知斐波那契数列（Fibonacci sequence），过去几个世纪数学家对其进行了深入研究。然而，人们不熟悉的是Tribonacci序列（Tribonacci sequence）。

Tribonacci序列由以下递推关系（recurrence）定义：

\[ \text{tri}(1) = 3 \quad (1) \]
\[ \text{tri}(n) = 1 + \frac{n}{2}, \quad \text{若 } n \text{ 为偶数} \quad (2) \]
\[ \text{tri}(n) = \text{tri}(n-1) + \text{tri}(n-2) + \text{tri}(n+1), \quad \text{若 } n \text{ 为奇数} \quad (3) \]

例如：

tri(2) = 1 + (2 / 2) = 2

tri(4) = 3

tri(3) = tri(2) + tri(1) + tri(4) = 2 + 3 + 3 = 8

给定非负整数n，需返回Tribonacci序列前n+1个数字的列表。

示例：tri(3) = [1, 3, 2, 8]
